# AGENTS.md

このリポジトリで作業するときの共通ルールです。

## ドキュメント更新

アップデートを行う場合は、同じ Pull Request で関連ドキュメントも更新します。

対象例:

- コード
- 設定
- 依存関係
- 仕様
- 権限
- データ構造
- 導入手順
- 運用手順

確認対象:

- README.md
- docs/SPECIFICATION.md
- docs/SETUP.md
- docs/OPERATIONS.md
- docs/TROUBLESHOOTING.md
- AGENTS.md

Pull Request には、更新したドキュメントの一覧を記載します。
